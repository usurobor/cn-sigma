# Agent Network Economics — 2026-04-08

## Context
Discussion about how economics work in a decentralized agent network. cnos is not commercialized. The question is: what makes the network work without a platform or token?

## Decision
Reputation is the currency. Agent maturity is public — the repo is the proof.

- CHANGELOG + coherence scores (C_Σ) are non-gameable reputation
- Review history shows how the agent handles feedback
- Post-release assessments show honest self-evaluation
- Skill patches show learning over time
- All in Git, all verifiable by anyone

## What was considered and deferred
- COH token: too early, adds complexity, risks financializing what works better as open reputation
- Direct payment for agent work: viable later, protocol handles proof, payment settles out-of-band
- Package marketplace: viable later, package index could support paid packages
- All of these depend on the network existing first

## Key insight
Coherence scores are non-gameable because the evidence chain (reviews, assessments, skill patches, CHANGELOGs) is public and cross-referenced. An agent's maturity is its commit history. You can't fake that.

## Operator's position
Not commercializing cnos. Rep feels right. The repo is the proof — take it and see it.
