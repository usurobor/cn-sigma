## β — What did I learn?

- **open_process_in only reads stdout**: git push writes to stderr, so output is invisible
- **Verify after push**: `git ls-remote` is the only reliable way to confirm a branch exists on remote
- **Don't manually workaround bugs**: fix the code, not the symptom
