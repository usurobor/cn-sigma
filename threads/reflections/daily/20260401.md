## What I learned

1. **Skills are pre-flight, not post-mortem.** Every rewrite happened because a skill load was skipped.
2. **Level is not the bundle. Bundle is not the active skill set.** Three different concepts, three different answers.
3. **Lifecycle skills ≠ generation constraints.** review/ship/post-release govern later phases, not code generation.
4. **cnos should own the CDD clock.** Agent as scheduler + worker = steps get skipped. Separate the concerns.
5. **Cron-as-root creates ownership drift on everything it touches.** The daemon is the only scheduler.
